# `mocks/` — design mockups, throwaway prototypes, brand experiments

Static HTML / SVG / image mockups that inform plans but are not production
code. Each mockup either references its source plan (`PLAN-*.md`) or stands
alone as a brand / UI exploration.

If a mockup tests a layout / routing rule that affects production code, also
add a worked-example SVG next to the plan in `doing/`.
